# Server operations

## Server setup
```
npm install
```

## Start server
```
npm start
```

_All other operations are run from the client directory_

# Client operations
Go to the client dir..
```
cd client
```

## Client project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
